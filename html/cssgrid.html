grid-template-columns: repeat(auto-fit, minmax(min(500px, 100%), 1fr)); 

display: grid;
grid-template-columns: repeat(auto-fit, minmax(min(500px, 100%), 1fr));

To prevent overflow your grid layout, and have a responsive grid layout.

auto-fit - tell the browser to handle the column sizing and element wrapping for us so that the elements will wrap into rows when the width is not large enough to fit them in without any overflow.

minmax and min for not overflowing.

This is very effective to basic grid layout.
