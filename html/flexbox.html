Index Flexbox

flex: 1 1 0;

/* Flexbox Method */
/* Parent Element */
display: flex;

/* Child Element */
flex: 1 1 0;
/* or */
flex-grow: 1;
flex-shrink: 1;
flex-basis: 0;

Use flex: 1 1 0; in child-flex-element to have same width element.

To allow a child-flex-element to grow and shrink as needed, ignoring its initial size.
