Index Pseudo Elements

::first-letter

.class-name::first-letter {
  font-size: 2rem;
  color: red;
  font-weight: bold;
}

::first-line

.class-name::first-line {
  color: red;
}

::selection

.class-name::selection {
  background-color: red;
  color: #fff;
}

We have already covered this pseudo element, but essentially it lets you style the highlighted content on page. Again you can only use certain CSS properties. Pretty much only properties that modify the color or text-decoration are allowed. You can highlight this paragraph for an example.
